iran-adblock
Ad blocking rules for persian websites. Works with Brave adblock import source (iOS,Windows,MacOS).

Installation
Brave (iOS,Windows,MacOS) under Setting -> Brave Shields & Privacy -> Content Filtering -> Add Custom Filter List
